OpenGL Core
Last modified February 25, 2015

This project is a simple set of utlities I use in my OpenGL projects.
Uses LWJGL 3.

Supports the following OpenGL versions:
* Archaic ( OpenGL 1.0 )
* Old ( OpenGL 2.1 )
* New ( OpenGL 3.0 )
* Modern ( OpenGL 4.2 )

Current features:
* Display management
* Loop handlers:
* * Rendering
* * Updates
* * Background Updates
